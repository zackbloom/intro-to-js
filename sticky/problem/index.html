The idea is to make the right bar (which has class "right") to be 'sticky'.

Sticky refers to the idea that it scrolls with the page normally, except if it would be scrolled off screen, it switches to be fixed to the top of the viewport. Take a peak at the solution to get an idea what I mean.

It's a useful technique when you have a bar of info or controls on a long page.

What you need

document.querySelector
Access an element on the page using a 'css-like' selector
document.body.scrollTop
How far down the page has been scrolled in pixels
document.addEventListener('scroll', fn)
Run a function whenever the page is scrolled
anyElement.style.position = 'fixed'
Makes the element position fixed

Bonus points

  • Instead of making the element position fixed in javascript, add a CSS class which specifies it.
  • Support the height of the nav bar changing.
anyElement.clientHeight
The height of an element (optional)
anyElement.classList.add
Add a class to an element
anyElement.classList.remove
Remove a class from an element
