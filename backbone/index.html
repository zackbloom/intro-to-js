Using an MVC, like Backbone, allows us to build bigger, more complex applications on the client.

To learn Backbone, you're going to make some changes to our Todo list.

I'd like to be able to

  • Delete items
  • Edit items
  • Reorder items

Hints

To delete an item, you want to call model.delete(). First though, you want to add some sort of delete button to each ListItemView.

To edit an item, you'll want to copy it's title into the input, and then change the title of that item when enter is pressed.

To support reordering, give each model a 'priority' attribute, and give the collection a comparator of that priority.

The Backbone Docs are your friend.
