Let's build a GitHub Search UI!

GitHub has a wonderful API. Start by grabbing an auth token, and adding it to the end of the url of this page. I have a bit of code in token.js which grabs it out of that hash, and makes it available for our search code, but don't worry about it now.

Once that's done, you should see a list of bitcoin projects below the search box. Unfortunately, the content of the search box is irrelevant right now, no matter what, we always search for bitcoin!

Your task is to change this page such that when I type into the search box, it actually searches for what I ask for.

We're going to be making requests to the Github API. I don't want you to get lost in the weeds of how that works just yet, so I've written the actual request-y bit, and packaged it up into a nice little function for you. You can just call search with your search term and a function you'd like it to run with the list of results.

Put all of your code in js/github.js.

What you'll need

anyInput.value
The current value of the input
anyInput.addEventListener('keyup', fn)
Runs fn whenever a key is pressed in the input
document.querySelector('.search')
Return the first element on the page with class 'search'
